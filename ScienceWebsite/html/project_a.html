Programming The Project


Project description: The lightwaves project is about programming a 2D model of the refraction of light using Snell's law. This required us to study and understand the physics behind the problem, and later apply it through our program.

This was my favourite project this year as I got to work with my friend from another class and I got to do something I love... coding.

To the left is all of the code used in the project. Feel free to click through the files to see how it works. It's all programmed in C# using Unity. Alternatively, you can look at the code in this GitHub repo. Unrelated: Please follow my GitHub account...
